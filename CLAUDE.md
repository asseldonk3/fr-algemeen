# Proeve van Bekwaamheid - ASML Illumination Module XT:860

## Project Doel

Dit project ondersteunt het afstudeerexamen (Proeve van Bekwaamheid) voor de opleiding **MBO Technicus Mechatronica** (Crebo 25893), niveau 4, BBL-leerweg bij **Summa College**.

### De PvB-opdracht
**Onderwerp:** Opbouwen en kwalificeren van de illumination module van een ASML TWINSCAN XT:860 machine (860 module buildmachine illumination).

### BPV-bedrijf
ASML (Advanced Semiconductor Materials Lithography)
