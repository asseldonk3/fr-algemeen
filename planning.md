# Planning Proeve van Bekwaamheid

## Overzicht alle taken

Dit document bevat alle taken die uitgevoerd moeten worden voor de Proeve van Bekwaamheid, gegroepeerd per fase.

---

## Fase 1: Voorbereiding (voorwaarden op orde)

- [ ] Theorie- en practicumopdrachten op schema
- [ ] BPV-beoordeling minimaal voldoende
- [ ] BPV-beoordelingsformulieren ingeleverd
- [ ] Alle projecten voldoende
- [ ] Goedkeuring van loopbaanbegeleider om te starten

## Fase 2: Oriëntatie op de opdracht

- [ ] Kwalificatiedossier doorlezen en begrijpen
- [ ] Beoordelingsformulieren B1-K1, P3-K1, P3-K2 bestuderen
- [ ] Alle kerntaken en werkprocessen doorlopen
- [ ] Checken of alle werkzaamheden uitvoerbaar zijn bij ASML
- [ ] Overleg met praktijkbegeleider over de opdracht (illumination module build + kwalificatie)
- [ ] Korte omschrijving van de opdracht schrijven
- [ ] Korte omschrijving inleveren in Canvas

## Fase 3: Organisatie

- [ ] Account student aangemaakt in EsMEI (door loopbaanbegeleider)
- [ ] Check: praktijkbegeleider heeft account in EsMEI
- [ ] Check: praktijkbeoordelaar heeft account in EsMEI
- [ ] Praktijkbeoordelaar heeft online training gevolgd op praktijkbeoordelaar.nl
- [ ] Kopie certificaat online training inleveren in Canvas
- [ ] Taak afvinken in Canvas

## Fase 4: Definitieve opdrachtomschrijving

- [ ] Word-document 'Formulier Opdracht Proeve van Bekwaamheid' downloaden
- [ ] Document invullen:
  - [ ] Eerste pagina volledig invullen
  - [ ] Titel van de opdracht
  - [ ] Tijdsduur + start- en einddatum
  - [ ] Opdrachtomschrijving (alle kerntaken en werkprocessen benoemen)
  - [ ] Rode tekst vervangen door eigen tekst
  - [ ] Schuingedrukte rode tekst verwijderen
- [ ] Document uploaden in EsMEI bij 'Opdrachtomschrijving'
- [ ] Accordering door praktijkbegeleider
- [ ] Accordering door Summa begeleider
- [ ] Accordering door student (LET OP: hierna kan niets meer gewijzigd worden!)
- [ ] Examencommissie stelt opdracht vast

## Fase 5: Uitvoering van de proeve

- [ ] Start met opbouw illumination module volgens planning
- [ ] Tijdens uitvoering bewijsmateriaal verzamelen:
  - [ ] Foto's van werkzaamheden
  - [ ] Ingevulde checklists / testlijsten
  - [ ] Tekeningen en schema's die je gebruikt hebt
  - [ ] Eventuele afwijkingsrapporten
  - [ ] Notities over werkprocessen
- [ ] Afspraak maken met schoolbeoordelaar voor observatiebezoek (per mail)
- [ ] Observatiebezoek: schoolbeoordelaar + praktijkbeoordelaar aanwezig
- [ ] Tijdens observatiebezoek datum eindgesprek vastleggen
- [ ] Kwalificatie van de module uitvoeren en documenteren
- [ ] Taak afvinken in Canvas

## Fase 6: Projectmap en eindpresentatie

### Projectmap (1 document)
- [ ] Duidelijke inhoudsopgave maken
- [ ] Per werkproces een hoofdstuk met bewijslast:
  - [ ] B1-K1-W1: Voorbereiding (werkorders, tekeningen, materiaalllijsten)
  - [ ] B1-K1-W2: Vervaardiging (foto's, beschrijving werkzaamheden)
  - [ ] B1-K1-W3: Plaatsing en controle samenbouw (checklists, foto's)
  - [ ] B1-K1-W4: Aansluitingen (schema's, testresultaten)
  - [ ] P3-K1 werkprocessen: _[aan te vullen]_
  - [ ] P3-K2 werkprocessen: _[aan te vullen]_
- [ ] Kop 1, kop 2 etc. correct gebruiken
- [ ] Werkproces-nummers in hoofdstuktitels (bijv. "B1-K1-W1: Voorbereiding")
- [ ] Check: is het overzichtelijk? Hoeven beoordelaars niet te puzzelen?
- [ ] Projectmap inleveren in EsMEI bij 'bewijsstukken'
- [ ] Indien nodig: 'Verklaring omtrent bedrijfsgeheim' invullen en inleveren

### Eindpresentatie
- [ ] PowerPoint maken met overzicht van het project
- [ ] Uitleggen wat je gedaan hebt
- [ ] Per werkproces toelichten hoe je dit hebt uitgevoerd
- [ ] Presentatie inleveren in EsMEI

### Canvas
- [ ] Taak afvinken in Canvas

## Fase 7: Afronding en eindbeoordeling

- [ ] Check: heeft praktijkbeoordelaar de observatiepunten in EsMEI ingevuld?
- [ ] Eindgesprek en presentatie op school (of afgesproken locatie)
- [ ] Presentatie geven aan beoordelaars
- [ ] Vragen van beoordelaars beantwoorden
- [ ] Beoordelaars vullen individuele beoordeling in EsMEI in
- [ ] Gezamenlijke beoordeling door beoordelaars
- [ ] Resultaat ontvangen
